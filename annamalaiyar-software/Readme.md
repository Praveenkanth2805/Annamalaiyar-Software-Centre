# Annamalaiyar Software Centre

A complete web application for software selling and teaching with admin management panel.

## Features

### Customer Features
- Browse products and courses without login
- Place orders directly
- Live price calculation
- Multi-language support (English/Tamil)
- Mobile responsive design

### Admin Features
- Secure admin login with bcrypt hashing
- Auto logout after 30 minutes inactivity
- Full CRUD operations for products and courses
- Order management with status tracking
- Customer management
- Sales reports and analytics
- Backup system
- Export to Excel functionality

## Technology Stack
- **Frontend:** HTML5, CSS3 (Material Design), JavaScript
- **Backend:** Python Flask
- **Database:** MySQL
- **Security:** Bcrypt + Salted passwords, Session-based auth
- **Hosting:** PythonAnywhere ready
